Theory

for this type of events we have prop: relatedTarget that complements target prop

for mouseover

event.target is the element that the cursor go in

event.relatedTarget is the element that the cursor go out

for mouseout

event.target is the element that the cursor go out

event.relatedTarget is the element that the cursor go in

difference between (mouseenter & mouseleave) and (mouseleave & mouseover)

  • transitions inside elements, into its children and from them are not counted
  • mouseenter & mouseleave are not bubbling
