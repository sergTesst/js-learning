Mutation observer

let observer = new MutationObserver(callback)

observer.observe(none, config)

config is an object with boolean values

  • childList changes in the children
  • subtree
  • attributes
  • attributeFilter array of attributes, in order to observe for chosen
  • characterData observe only for node.data (text data)
  • characterDataOldValue if true old & new value into the callback, default false and pass new value
  • attributeOldValue same as characterDataOldValue for attributes

then after changes, callback is executed, in witch changes are passed as a first arg
as a list of obj with type MutationRecord, and observer as a second arg

Props of MutationRecord

  • type - type of change

    • "attributes" attr is changed
    • "characterData" elem.data changed, is used for text node
    • "childList" add/remove child nodes
  • target - where change has happened:
    elem for "attributes"
    text node for "characterData"
    elem for "childList"
  • addedNodes/removeNodes add/remove nodes
  • previous/nextSibling previous or next same level element for add/remove elements
  • attributeName/attributeNamespace - name of namespace (for XML) changed attribute
  • oldValue - previous value, only for changed attribute or text, if corresponding option attributeOldValue/characterDataOldValue is turned on
