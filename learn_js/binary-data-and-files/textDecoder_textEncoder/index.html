TextDecoder & TextEncoder

let decoder = new TextDecoder([label], [options]);

  • label type of encoding, utf-8 - default, supported(big5, windows-1251) and more

  • options

    • fatal - boolean, if true throws an error for invalid symbols, otherwise replaced width sybmol \uFFFD
    • ignoreBOM -- boolean

let str = decoder.decode([input], [options]);

  • input - binary buffer

  • options

    • stream
